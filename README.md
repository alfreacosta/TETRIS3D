# TETRIS 3D - OpenGL - Alfredo Acosta 
## Descripcion del Proyecto

Tetris 3D es un video juego  desarrollado en Python utilizando PyOpenGL y NumPy. El objetivo es jugar Tetris en un tablero tridimensional, rotando y moviendo piezas en los tres ejes.

## Lenguajes y tecnologias
- Python 3.8+
- PyOpenGL
- NumPy

## Requerimientos
- Python 3.8 o superior
- PyOpenGL >= 3.1.5
- NumPy >= 1.19.0
- Pygame >= 2.0.0
## Instalacion

```bash
pip install -r requirements.txt
python3 main.py
```

## Controles
### Menu
- **1, 2, 3**: Seleccionar nivel
- **ENTER**: Iniciar juego
- **ESC**: Salir

### Juego
**Movimiento:**
- **Flechas**: Mover pieza (izquierda, derecha, adelante, atras)
- **ESPACIO**: Caida rapida

**Rotacion:**
- **Q / W / E**: Rotar eje X -90, 180, +90
- **A / S / D**: Rotar eje Y -90, 180, +90
- **Z / X / C**: Rotar eje Z -90, 180, +90

**Camara:**
- **2, 4, 6, 8**: Vistas predefinidas
- **5**: Vista inicial
- **+ / -**: Zoom

**Otros:**
- **P**: Pausar
- **M**: Modo automatico (IA)
- **R**: Reiniciar juego
- **O**: Activar/Desactivar musica

## Caracteristicas

- Tablero 3D
- Rotacion en los tres ejes
- Sistema de iluminacion con dos fuentes de luz
- Multiples vistas de camara
- IA que juega automaticamente
- Tres niveles de dificultad
- Deteccion de colisiones en 3D
- Sistema de puntuacion
